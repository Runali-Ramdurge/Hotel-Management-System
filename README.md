🏨 Hotel Management System (C++)
📌 Overview

This is a console-based Hotel Management System developed in C++. It helps manage hotel room bookings, customer records, and billing using Object-Oriented Programming (OOP) and file handling.

🚀 Features

➕ Book a Room
📋 View Customer Record
🏨 Display Allotted Rooms
✏️ Modify Customer Details
❌ Delete Customer Record
💰 Automatic Fare Calculation
💾 Store & Retrieve Data from File

🛠️ Tech Stack

Language: C++

Concepts Used:

Classes & Objects
File Handling (fstream)
OOP Concepts
Conditional Statements
Functions
📂 File Storage

⚙️ How to Run
🔹 Using Dev C++
Open Dev C++
Create new file
Paste code
Save as main.cpp
Press F11 (Run)
🔹 Using Terminal
g++ main.cpp -o hotel
./hotel
